# API Endpoints

- `GET /api/v1/health` — API health
- `GET /api/v1/layers` — List layers
- `GET /api/v1/events` — List events (filters: bbox, layer, time, source, etc.)
- `GET /api/v1/events/{id}` — Event by ID (to be added)
- `GET /api/v1/sources` — List source providers/feeds
- `GET /api/v1/regions` — Region presets
- `GET /api/v1/search` — Keyword search
- `GET /api/v1/translations/{event_id}` — Translations for an event

OpenAPI: `/api/v1/openapi.json`
